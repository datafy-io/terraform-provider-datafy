# datafy_role_arn (Data Source)

Use this data source to retrieve the AWS IAM role ARN associated with a Datafy account. This is useful when you need to reference an IAM role association that was configured outside of Terraform.

## Example Usage

```terraform
data "datafy_role_arn" "example" {
  account_id = "79c406c5-7b64-43f2-ba76-9b01e74e3d90"
}

output "associated_role" {
  value = data.datafy_role_arn.example.arn
}
```
