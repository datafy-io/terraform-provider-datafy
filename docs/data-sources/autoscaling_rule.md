# datafy_autoscaling_rule (Data Source)

Use this data source to retrieve information about an existing Datafy autoscaling rule. This is useful when you need to reference a rule that was created outside of Terraform or in a different Terraform configuration.

## Example Usage

```terraform
data "datafy_autoscaling_rule" "example" {
  account_id = "79c406c5-7b64-43f2-ba76-9b01e74e3d90"
  rule_id    = "rule-01KB3R5TBDA729KK95KJ72JWS5"
}

output "rule_active" {
  value = data.datafy_autoscaling_rule.example.active
}

output "rule_policy" {
  value = data.datafy_autoscaling_rule.example.rule
}
```

## Schema

### Required

- `account_id` (String) The unique identifier of the Datafy account that owns the rule.
- `rule_id` (String) The unique identifier of the autoscaling rule to look up.

### Read-Only

- `active` (Boolean) Whether the autoscaling rule is currently active.
- `rule` (String) The autoscaling rule policy as a JSON string.
